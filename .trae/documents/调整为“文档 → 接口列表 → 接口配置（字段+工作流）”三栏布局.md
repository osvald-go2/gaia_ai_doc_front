## 布局目标
- 三列并排，顺序为：
  1) 文档内容（只展示文档）
  2) 接口列表（只展示列表）
  3) 接口配置（字段配置 + 工作流预览 + SQL 面板）
- 每列支持：拖拽调整宽度、收起/展开；收起显示简略内容（图标/数量/标题），拖拽后自适应。

## 技术实现
- 使用 `react-resizable-panels` 构建三列：`PanelGroup (horizontal)` → 三个 `Panel` → `PanelResizeHandle`。
- 三列容器统一：`flex flex-col h-full min-h-0 bg-card border-r border-border`；内部滚动用 Radix `ScrollArea`（`Viewport h-full`）。
- 高度分配：整页 `h-screen flex-col`，三列内部 `flex-1 min-h-0`，防止内容挤压。
- 尺寸持久化：将三列比例存储到 `localStorage`，默认比例：文档 30%，列表 30%，配置 40%。

## 交互与收起样式
- 每列的“展开态”与“收起态”切换：
  - 展开态：正常内容区域（文档卡片/列表/编辑器）。
  - 收起态：固定宽度（如 56px）细栏，居中显示图标与简要信息：
    - 文档列：`FileText` 图标 + 竖排“文档”。
    - 列表列：`Code2` 图标 + 竖排“接口”。
    - 配置列：`Settings2` 图标 + 竖排“配置”。
  - 点击收起栏区域可展开；把手支持拖拽；双击把手重置尺寸。

## 列内职责与联动
- 文档列：仅渲染 `DocumentPanel`；保留章节锚点与点击跳转；不混入列表内容。
- 列表列：仅渲染 `InterfaceList`；选择列表项时更新“配置列”的选中接口。
- 配置列：渲染 `InterfaceEditor`（字段配置视图/工作流预览视图切换）；保留底部 SQL 面板。
- 联动：
  - 点击文档某段落 → 触发 `onSectionClick(sectionId)` → 列表列高亮对应接口并滚动到视图 → 配置列切换到该接口。
  - 列表项选择 → 配置列加载对应接口。

## 响应式与断点
- 桌面（≥ 1280px）：三列按默认比例显示；拖拽后比例保留。
- 平板（≥ 1024px 且 < 1280px）：允许任意列收起；默认稍微加大“配置列”（45%）。
- 移动（< 1024px）：默认仅显示“配置列”，文档/列表以抽屉形式从左右滑出（保留后续扩展接口）。

- 文档与列表联动：点击文档段落切换列表与配置；选择列表项切换配置。
